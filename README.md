# 🏠 Rent & Flatmate Finder

An AI-powered web application that helps users find rental properties and compatible flatmates based on lifestyle preferences.

## 📌 Features

- User Registration & Login
- Create & Manage Property Listings
- Search & Filter Listings
- Flatmate Profile Management
- AI Compatibility Score using Google Gemini
- Chat System between Users
- Responsive Dashboard

## ⚙ Installation

### Clone Repository

```bash
git clone <https://github.com/TheAkshatGupta/Rent-Flatmate-Finder>
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Backend

```bash
cd backend
python run.py
```

### Run Frontend

```bash
cd frontend
python -m http.server 5500
```

## 🤖 AI Compatibility

The application uses Google Gemini AI to analyze user preferences such as budget, city, lifestyle, and housing preferences to generate a compatibility score with a natural language explanation.
